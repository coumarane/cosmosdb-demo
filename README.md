# cosmosdb-demo
Must have an Azure account

Create a appsettings.json file if not exists
and add the cosmos db settings

```
{
    "accountEndpoint": "https://xxx.documents.azure.com:443/",
    "authKeyOrResourceToken": "xx"
}
```

# vscode debug config
**tasks.json**
```
{
    "version": "2.0.0",
    "tasks": [
        {
            "label": "build",
            "command": "dotnet",
            "type": "process",
            "args": [
                "build",
                "${workspaceFolder}/cosmosdb.csproj",
                "/property:GenerateFullPaths=true",
                "/consoleloggerparameters:NoSummary"
            ],
            "problemMatcher": "$msCompile"
        },
        {
            "label": "publish",
            "command": "dotnet",
            "type": "process",
            "args": [
                "publish",
                "${workspaceFolder}/cosmosdb.csproj",
                "/property:GenerateFullPaths=true",
                "/consoleloggerparameters:NoSummary"
            ],
            "problemMatcher": "$msCompile"
        },
        {
            "label": "watch",
            "command": "dotnet",
            "type": "process",
            "args": [
                "watch",
                "run",
                "--project",
                "${workspaceFolder}/cosmosdb.csproj"
            ],
            "problemMatcher": "$msCompile"
        }
    ]
}
```

            // If you have changed target frameworks, make sure to update the program path.
